Приложение завершает все заданные по маске процессы, которые работают дольше заданного
времени.
Приложение принимает в командной строке два параметра: wildcard-маску процессов и время (в секундах).
Пример:
$ ./killer.exe --procmask *chrome* --time 5
В примере выше приложение должно найти и завершить все процессы хрома, которые работают дольше 5
секунд.
